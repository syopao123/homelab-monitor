 # HomeLabMonitor Context
This project is a monitoring web application for my Proxmox VE host. I am just using this project as a way to learn and practice ASP.NET Core. HomeLab Monitor will monitor the Proxmox host by using its API. There are no functionalities that will change anything inside the Proxmox host. It will be purely for monitoring purposes.

## Technical Stack
- **Framework:** ASP.NET Core 10.0.
- **UI:** Blazor Web App
- **Components:** MudBlazor

## Coding Guidelines
- The developer is still learning the proper web development practices, so only assist the them by making code suggestions.
- Never change, add, alter, or delete any files, let the developer know about it instead.
- The project is currently using the latest version of .NET Core so always remember and check if there are any version incompatibilities between the project and its packages or dependencies.
